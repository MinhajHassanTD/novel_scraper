# Novel Scraper

A simple Python tool to scrape and download novels from novelbin.com.

## Features

- Fetches chapters sequentially
- Extracts chapter titles and content
- Saves to a single text file
- Automatic navigation to next chapters

## Usage

```bash
python fetch_webpage.py
```

Modify the `current_url` and chapter range in `fetch_webpage.py` as needed.

## Requirements

- beautifulsoup4
- cloudscraper
